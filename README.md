# Layoffs-Data-Cleaning-EDA
# Layoffs Data Cleaning & Exploratory Data Analysis

## Project Overview

This project focuses on cleaning and analyzing a global layoffs dataset using SQL.

The objective was to prepare raw data for analysis and identify trends in layoffs across industries, companies, countries, and time periods.

## Skills Demonstrated

- SQL Data Cleaning
- Duplicate Removal
- Data Standardization
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Aggregations
- Window Functions
- Working with CTES

## Dataset

Global layoffs dataset containing information on:

- Company
- Industry
- Country
- Date
- Total Laid Off
- Percentage Laid Off
- Funds Raised

## Project Structure

```text
data/
    layoffs.csv

sql/
    Data_Cleaning.sql
    Exploratory_Data_Analysis.sql
```

## Data Cleaning Tasks

- Removed duplicate records
- Standardized text fields
- Handled null values
- Corrected inconsistent formatting
- Prepared data for analysis

## Exploratory Analysis

Analysis included:

- Layoffs by company
- Layoffs by industry
- Layoffs by country
- Layoffs by year
- Rolling totals
- Trend analysis

## Key Findings

- Certain countries and industries were disproportionately affected.
- Layoffs peaked in 2023
